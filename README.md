To run the dev server for your app, use:

```sh
npm run start-csr
```

or server side rendering

```sh
npm run start
```

To create a production bundle:

```sh
npm run build-csr
```

or server side rendering

```sh
npm run build
```

To see all available targets to run for a project, run:

```sh
npx nx show project angular-youtube
```
